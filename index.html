Google Integration

⚙️ One-time Setup Required

To use Google services, you need a free OAuth Client ID from Google Cloud Console. This takes about 5 minutes and is free.

  1. Go to console.cloud.google.com
  2. Create a project → APIs & Services → Credentials
  3. Click Create Credentials → OAuth Client ID → Web application
  4. Add http://localhost and your file path under Authorized JavaScript origins
  5. Enable: Google Drive API, Google Calendar API, Gmail API, Google Sheets API
  6. Copy your Client ID and paste it below
